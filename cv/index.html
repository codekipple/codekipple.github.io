CSS architecture

I am extremely interested in CSS architecture. It's surprising just how difficult CSS architecture can be, particularly on larger sites/apps which have multiple developers adding and editing CSS.

I am constantly working towards improving my CSS architecture. I currently use a lot of ideas i have gleaned from Nicole Sullivan, Nicolas Gallagher and Harry Roberts.

I would describe my current CSS architecture as OOCSS with a BEM like naming convention.
